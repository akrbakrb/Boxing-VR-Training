## 🧠 Overview

🔴 VR boxing training simulation built in Unity
🟢 Focus on realistic punch interaction and training mechanics
🔵 Designed as a practice/demo environment for VR development

## 🚀 Features

🔴 **VR Punch Interaction**
🟢 Real-time hand tracking with VR controllers
🔵 Physics-based punch detection

🔴 **Training Equipment**
🟢 Heavy bag for power punches
🔵 Speed bag for timing and rhythm
🔴 Double-end bag for accuracy training

🔴 **Gym Environment**
🟢 Minimal boxing gym setup
🔵 Training props (weights, rack, equipment)

🔴 **Feedback System**
🟢 Hit detection on targets
🔵 Basic visual/physics response

## 🛠️ Built With

🔴 Unity (Tested on Unity 6+)
🟢 XR Interaction Toolkit / OpenXR
🔵 C#

## ⚙️ Getting Started

🔴 Clone the repository
🟢 Open in Unity Hub
🔵 Load main scene (e.g., BoxingTrainingScene)
🔴 Connect VR headset (Meta Quest / PCVR)
🟢 Press Play ▶️

## 🎮 Controls

🔴 Punch – Move controllers forward
🟢 Grab (if enabled) – Trigger / Grip
🔵 Move – Joystick / Thumbstick

## ⚠️ Known Issues

🔴 Physics may feel inconsistent on fast punches
🟢 Limited feedback (no sound/haptics yet)
🔵 Not optimized for performance
